# Company Interview Wiki
야근을 하던 중 동료가 물어봤다.
- 💁🏻‍ "이 회사가 몇 점이라고 생각하세요?"
- 👩🏻‍💻 "음... 대충 6점?"
- 💁🏻‍ "왜요? 기준이 있을 거 아니에요."
- 👩🏻‍💻 "그러게요...?"

그래서 생각해봤다.
- 나는 어떤 항목들로 회사를 평가하는가?
- 나의 가치관이 충분히 반영되었는가?
- 항목별로 가중치는 어떻게 부여할까?
- 평가점수를 나의 만족도라고 칭할 수 있을까?

그리고 궁금해졌다.
- 다른 사람들은 무엇을 중요하게 여길까?
- 다들 본인의 가치관에 맞는 회사를 다니고 있을까?
- 다른 사람들의 만족도는 어떨까?
- 가중치와 평가점수(만족도)를 어떻게 해석할 수 있을까?
- 우리의 해석을 통해 다른사람들에게 긍정적인 가치를 전달할 수 있을까?
